MIS PROJECT DOCUMENTATION

This page covers the documentation of the features provided in this Management Information System (MIS) software. This software is used for taking live updates on data from the Solar plants at Beawar, Jharkhand, Castamet, Roorkee and Panipat. This software provides the following features:

  1. Latest Statistics
  2. Analysis Window
  3. Solar Exception Report
  4. Getting Started
  5. Operator Login



Refer the sections below to see how to use these features.

Latest Statistics

This provides you with the latest information about all the solar plant sites, one by one. These data are categorised in vertically collapsing sections. Each section is identified by the site name followed by the number of invertors at that site.
For each site, the following particulars are displayed:

  1. Daily Solar Generation (in kWh)
  2. Daily Solar Generation (in % PLF)
  3. Daily Performance Ratio (in %)
  4. Inclined irradiance (in kWh/m2)
  5. Bar Graph representing the latest data about the solar generations from all the invertors

To access this page, click on localhost:8000/sites-liveupdates/



Analysis Window

This provides you the space for data analysis. Plug into the required type of analysis. Two options are made available for analysis:

  • Analyse Solar Data At A Particular Site
  • Analyse Solar Data Between Different Sites

Analyse Solar Data At A Particular Site
In this mode of analysis, you need to select a few parameters based upon which tables and graphs will be generated.
Select Date, Site and two invertors to compare their solar generation.
You will also be presented with that day's solar generation statistics from the site.

Analyse Solar data between different sites
This mode of analysis allows to compare the data between different sites.
Select a Date and the Particular from the form that shows up.
All the sites are compared for that particular for the specific date, and the results are presented in the tabular form along with the graphs.

To access this page click on localhost:8000/analysis-window/



Solar Exception Report

This page presents with the Day's Solar Exception Report. This report comprises of all the figures/metrics that are calculated from the everyday's Solar Data Sheets and WMS Reports.
Select a date to get the finalised Solar Exception Report for that day.
There is also an option for downloading this report in Excel Sheet format, which can be used for many different and useful purposes.

To access this page click on localhost:8000/exception-report/



Getting Started

This page presents the brief documentation of the software. Click on the links to know more about the feature.

To access this page click on localhost:8000/getting-started/



Operator Login

This pairs up to the Operator Login Page. This is accessible only to the Site Operators and Upper Management Staff.
By logging into the admin page, operators can see and edit the compiled information particulars of each and every site.
The procedure of adding some Target Data about every particular site has to be followed everyday, because the final compiled results do depend on the Targets set by the Operators.

To access this page click on localhost:8000/admin/
